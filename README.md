# Experimental Space Law LLM Wiki

This is an experimental [LLM wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) about space law. It attempts to conform to the [Open Knowledge Format](https://cloud.google.com/blog/products/data-analytics/how-the-open-knowledge-format-can-improve-data-sharing), introduced by Google in June 2026.

I generated this from a collection of statutes, regulations, executive orders, and treaties about space law.

It is **not** complete. **Do not** rely on it! But **do** have fun.
